# SiteOS — India's Construction OS

A role-based **Construction Project Management SaaS** built with **Flutter + Supabase**, designed for Indian mid-size builders managing 5–20 active sites.

## Tech Stack

- **Flutter 3.32** — iOS, Android, Web
- **Supabase** — Auth, PostgreSQL, Edge Functions, Storage
- **Firebase** — Crashlytics, Hosting (Web)
- **Gemini AI** — Free-tier AI features

See [runbook.md](runbook.md) for setup instructions.
